# Client-Side File Converter

A powerful, secure, and privacy-focused web-based file converter that runs entirely in your browser. No files are ever uploaded to a server‚Äîeverything is processed locally on your device.

*   **JPEG to PDF**: Convert multiple image files into a single PDF document.
*   **PDF to JPEG**: Convert PDF pages into high-quality JPEG images.
*   **PDF to Word**: Extract text from PDF files and convert them into editable Word documents (`.docx`).
*   **Word to PDF**: Convert Word documents (`.docx`) into PDF format preserving text layout.
*   **Merge PDF**: Combine multiple PDF files into a single document in your specified order.

## üì¶ Installation & Usage

1.  **Clone or Download** this repository.
2.  **Open `index.html`** in any modern web browser (Chrome, Firefox, Edge, Safari).
3.  **Select a Feature** from the tabs (e.g., "Merge PDF").
4.  **Drag & Drop** or **Browse** for your files.
5.  Click the **Convert/Merge** button to process and download your file instantly.
